# アイコンの配置場所

このフォルダに、次のファイル名でアイコン（PNG）を置いてください。

## 必須（最低限これがあれば動作します）
- icon-192x192.png  … メイン（Android・Apple用）
- icon-512x512.png  … 高解像度・スプラッシュ用

## 推奨（各種デバイス用）
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png  （iPad用）
- icon-384x384.png

## アイコンが1枚だけある場合
512x512 の画像が1枚ある場合：
1. icon-512x512.png としてこのフォルダに保存
2. 画像編集ソフトやオンラインツールで上記サイズにリサイズして、同じ名前で保存

例：Canvaで「ダウンロード」→ サイズを変えて再度ダウンロード、または irfanview / Photopea などでリサイズ。
